---
title: "Get Profile"
api: "GET /api/connections/{id}/me"
description: "Retrieve the WhatsApp profile information for a connection."
---

Returns the WhatsApp profile information (phone number ID and display name) for the linked account on this connection. The connection must be in `online` status.

## Authentication

<ParamField header="Authorization" type="string" required>
  Bearer token from Supabase Auth. Format: `Bearer <token>`
</ParamField>

## Path Parameters

<ParamField path="id" type="string" required>
  The unique identifier of the connection.
</ParamField>

<ResponseExample>
```json 200
{
  "id": "1234567890@c.us",
  "pushName": "John Doe"
}
```

```json 401
{
  "statusCode": 401,
  "message": "Unauthorized"
}
```

```json 404
{
  "statusCode": 404,
  "message": "Connection not found"
}
```
</ResponseExample>
