---
title: "List Chats"
api: "GET /api/connections/{id}/chats"
description: "Retrieve recent WhatsApp chats for a connection."
---

Returns the most recent WhatsApp chats (up to 20) for the specified connection. The connection must be in `online` status for this endpoint to return results.

## Authentication

<ParamField header="Authorization" type="string" required>
  Bearer token from Supabase Auth. Format: `Bearer <token>`
</ParamField>

## Path Parameters

<ParamField path="id" type="string" required>
  The unique identifier of the connection.
</ParamField>
