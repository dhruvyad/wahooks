---
title: "Update Webhook"
api: "PUT /api/webhooks/{id}"
description: "Update an existing webhook configuration."
---

Updates the specified webhook configuration. All body fields are optional -- only the fields you include will be updated. You can change the delivery URL, the subscribed events, or toggle the webhook on/off.

## Authentication

<ParamField header="Authorization" type="string" required>
  Bearer token from Supabase Auth. Format: `Bearer <token>`
</ParamField>

## Path Parameters

<ParamField path="id" type="string" required>
  The unique identifier of the webhook configuration to update.
</ParamField>

## Body Parameters

<ParamField body="url" type="string">
  The new HTTPS URL where webhook events will be delivered.
</ParamField>

<ParamField body="events" type="string[]">
  An updated array of event types to subscribe to.
</ParamField>

<ParamField body="active" type="boolean">
  Set to `false` to pause webhook deliveries without deleting the configuration. Set to `true` to resume.
</ParamField>
