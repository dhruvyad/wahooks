---
title: "Get Webhook Logs"
api: "GET /api/webhooks/{id}/logs"
description: "Retrieve delivery logs for a webhook configuration."
---

Returns the most recent delivery log entries (up to 100) for the specified webhook configuration. Each log entry records a single delivery attempt including the event payload, delivery status, and number of attempts.

## Authentication

<ParamField header="Authorization" type="string" required>
  Bearer token from Supabase Auth. Format: `Bearer <token>`
</ParamField>

## Path Parameters

<ParamField path="id" type="string" required>
  The unique identifier of the webhook configuration.
</ParamField>
