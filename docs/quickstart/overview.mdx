This guide walks you through connecting a WhatsApp number, receiving webhooks, and sending your first message.

## Prerequisites

- A WAHooks account ([sign up](https://wahooks.com/signup))
- A WhatsApp account on your phone
- An API token (create one in the dashboard or via CLI)

## 1. Install an SDK

<CodeGroup>
```bash TypeScript
npm install @wahooks/sdk
```

```bash Python
pip install wahooks
```
</CodeGroup>

## 2. Create an API token

<CodeGroup>
```bash CLI
wahooks login
wahooks tokens create my-app
```

```bash cURL
# Use your Supabase JWT from the dashboard
curl -X POST https://api.wahooks.com/api/tokens \
  -H "Authorization: Bearer YOUR_JWT" \
  -H "Content-Type: application/json" \
  -d '{"name": "my-app"}'
```
</CodeGroup>

Save the token — it's shown only once. It looks like `wh_a1b2c3...`.

## 3. Create a connection and scan QR

<CodeGroup>
```typescript TypeScript
import { WAHooks } from '@wahooks/sdk';

const client = new WAHooks({ apiKey: 'wh_...' });

const connection = await client.createConnection();
console.log('Connection ID:', connection.id);

// Get QR code to scan with WhatsApp
const qr = await client.getQR(connection.id);
console.log('Scan this QR:', qr.value);
```

```python Python
from wahooks import WAHooks

client = WAHooks(api_key="wh_...")

connection = client.create_connection()
print("Connection ID:", connection["id"])

# Get QR code to scan with WhatsApp
qr = client.get_qr(connection["id"])
print("Scan this QR:", qr["value"])
```
</CodeGroup>

Open the QR code in WhatsApp: **Settings → Linked Devices → Link a Device**.

## 4. Set up a webhook

<CodeGroup>
```typescript TypeScript
const webhook = await client.createWebhook(
  connection.id,
  'https://your-server.com/webhook',
);
console.log('Webhook created:', webhook.id);
console.log('Signing secret:', webhook.signingSecret);
```

```python Python
webhook = client.create_webhook(
    connection["id"],
    url="https://your-server.com/webhook",
)
print("Webhook created:", webhook["id"])
print("Signing secret:", webhook["signingSecret"])
```
</CodeGroup>

## 5. Send a message

<CodeGroup>
```typescript TypeScript
await client.sendMessage(
  connection.id,
  '1234567890@s.whatsapp.net',
  'Hello from WAHooks!'
);
```

```python Python
client.send_message(
    connection["id"],
    chat_id="1234567890@s.whatsapp.net",
    text="Hello from WAHooks!",
)
```
</CodeGroup>

<Note>
  Chat IDs use the format `{phone}@s.whatsapp.net` for individual chats
  and `{id}@g.us` for group chats.
</Note>
