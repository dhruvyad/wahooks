WAHooks lets you connect WhatsApp numbers to your application through a simple API. Send and receive messages, get real-time webhook notifications, and manage everything through the dashboard, CLI, or SDKs.

## How it works

<Steps>
  <Step title="Connect a WhatsApp number">
    Create a connection and scan a QR code to link your WhatsApp account.
  </Step>
  <Step title="Configure webhooks">
    Set up webhook URLs to receive real-time events when messages arrive.
  </Step>
  <Step title="Send and receive messages">
    Use the API, SDK, or CLI to send messages and process incoming webhooks.
  </Step>
</Steps>

## Key features

<CardGroup cols={2}>
  <Card title="Managed Infrastructure" icon="cloud">
    No need to host WAHA yourself. WAHooks handles provisioning, scaling, and health monitoring.
  </Card>
  <Card title="Webhook Delivery" icon="webhook">
    Reliable webhook delivery with HMAC-SHA256 signatures, exponential backoff, and delivery logs.
  </Card>
  <Card title="SDKs" icon="code">
    Official Python and TypeScript SDKs with full API coverage.
  </Card>
  <Card title="CLI" icon="terminal">
    Full-featured CLI for managing connections, webhooks, and API tokens.
  </Card>
</CardGroup>

<CodeGroup>

```bash pip
pip install wahooks
```


</CodeGroup>
